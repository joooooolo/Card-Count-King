This is Card Counting King

You can practice your card counting skills by playing the game at the same time

Once you start game by clicking Deal, you start counting the cards dealt on the table

  • Deal
  • Hit
  • Stay
  • Double
  • Split

At a random time during the game, you will be asked about the True Count

  1. Cards 2-6 = +1
  2. Cards 7-9 = 0
  3. Cards 10-A = -1

Blackjack Counting

If you answer correctly, you will continue with the same count, if not, cards will be shuffled and you will restart the game
